# Papersearch
Have 400+ photos as slides for your windows background? did someone ask for a specific picture on your slides? Well look no further You too can now on your very own copy of PAPERSEARCH! yea it just helps you find the current image displayed on your computers wallpaper. 

Wallpaper Finder – README
=========================

What this app does:
-------------------
This tool identifies the original source image for your current Windows wallpaper.
It works with both ZIP archives and folders full of images.

How to use:
-----------
1. Run WallpaperFinder.exe
2. On first launch, choose your image ZIP or folder
3. The app will index your images (this may take a few minutes)
4. The tray icon will appear near your clock
5. Right-click the tray icon for:
   - Show Window
   - Quick Locate & Copy
   - Refresh Now
   - Exit

Where data is stored:
---------------------
All settings and the image index are stored in:

%APPDATA%\WallpaperFinder\

This includes:
- wallpaper_index.db
- settings.json

Changing your image source:
---------------------------
Open the main window and click “Change Source”.
You can select a new ZIP or folder at any time.

Auto-refresh:
-------------
Enable auto-refresh to automatically detect wallpaper changes.

No installation required:
-------------------------
This app is fully portable. No Python or setup needed.
